# Icon Language

<!-- source-of-truth-standard: contract overrides markdown -->

Design Lab ikon dili ham lucide ikonlarini tek basina kullanmak yerine tasarlanmis container ve modul rengiyle sunar.

## Ilkeler

- Modul ikonlari container icinde gosterilir.
- Status ikonlari success, warning, danger ve info tokenlariyla ayrisir.
- Ikon stroke width tema konseptinin karakterine gore degisir.
- Executive Premium daha ince ve rafine ikon kullanir.
- Anatolian Modern daha sakin ve sicak container kullanir.
- Technical Command daha belirgin teknik container ve yuksek okunabilir status dili kullanir.

## Modul kapsami

Sirketler, ortaklar, temsilciler, subeler, muhasebe, IK, belgeler, sozlesmeler, Action Center ve Audit ikonlari Design Lab icinde karsilastirilir.
