# Theme Motif And Corner Art Guide

Date: 2026-06-08

Motifs are decorative identity helpers, not content. They should appear on page banners, featured cards, empty states and large summary panels. They should not sit in dense table rows or form inputs.

| Theme | Motif language |
| --- | --- |
| Classic | near-invisible frame/grid ticks |
| Art Deco Premium | thin stepped geometric corner lines |
| Anadolu 60'lar | retro sun/ray and warm rings |
| Green Atelier | botanical branch/leaf contour |
| Pop Studio | controlled graphic corner blocks and dots |

Light mode may show motifs more clearly. Dark mode should reduce contrast and avoid glow-heavy decoration.
