# Theme Coverage Matrix

| Area | Coverage |
| --- | --- |
| Shell/header/sidebar | Tokenized nav/header/background variables |
| Page banner | Theme background, radius, border, shadow and motif layer |
| Smart list/table | Toolbar/table/row/header variables and content motif layer |
| Cards/forms/buttons | Global token bridge and radius tokens |
| Badges/status | Semantic token colors |
| Wizards | Wizard token variables and global wizard bridge |
| Motifs | Theme-specific page and content CSS vector motif variables |
| Logo | Not changed, no recolor/filter |
