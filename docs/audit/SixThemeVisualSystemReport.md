# Six Theme Visual System Report

Implemented six canonical selectable themes: Hikmet, Bozkır, Eşitlik, Tabiat, Atlas and Avangard. Theme and appearance remain separate preferences. The root uses `data-eden-theme` and `data-appearance`, with CSS variables applied without changing the Eden logo.

Core coverage now targets shell, header, sidebar, page banner, smart list/list containers, tables, cards, forms, badges, wizard surfaces and motif/corner art through shared tokens and global bridge selectors.
