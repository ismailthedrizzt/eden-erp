# Accounting Known Gaps

Bu faz cari kartlar ve cari hareketler icin urun seviyesinde MVP temelidir.
Asagidaki konular bilincli olarak sonraki fazlara birakilmistir:

- Cift tarafli muhasebe ve yevmiye fisleri yok.
- e-Fatura/e-Arsiv entegrasyonu yok.
- Banka API entegrasyonu yok.
- Kredi karti hareketi API entegrasyonu yok.
- Belge OCR ve otomatik belge okuma yok.
- Otomatik mutabakat yok.
- KDV, stopaj, tevkifat ve vergi muhasebesi yok.
- Cari yaslandirma raporu yok.
- Nakit akis, gelir tablosu, bilanço gibi finansal raporlar yok.
- Sermaye odeme mutabakati paneli sadece data contract olarak hazir.
- Admin tarafindan ozellestirilebilir muhasebe hesap plani yok.
- Analitik ve failed reconciliation raporlari yok.
