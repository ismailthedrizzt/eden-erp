# Search Known Gaps

- Postgres full text search indexes are not yet productized.
- Typo tolerance/fuzzy search is not implemented.
- Semantic search and AI command parsing are future scope.
- Saved searches and search analytics are not available.
- Recent items do not yet run deep entity revalidation before display.
- Search metrics intentionally avoid raw query logging; analytics model is future.
- Cross-language stemming is not available.
- Advanced permission-aware indexing is not implemented.
- Provider timeouts are coarse MVP behavior; per-provider timeout tuning is future.
