# Organization / Facility E2E Checklist

## Teşkilat/Kadro

- [ ] Organizasyon ağacı açılır.
- [ ] Şirket filtresi çalışır.
- [ ] Aktif/pasif filtreleri çalışır.
- [ ] Organizasyon birimi oluşturulur.
- [ ] Parent değişikliği aynı şirket altında doğrulanır.
- [ ] Cycle oluşturma engellenir.
- [ ] Pozisyon/kadro eklenir.
- [ ] Planlanan/mevcut kadro summary güncellenir.
- [ ] Şube Açılışı ile oluşan unit görünür.
- [ ] Unit detayında ilişkili şube paneli görünür.
- [ ] Organization scope temsil yetkileri read-only görünür.

## Tesisler/Lokasyonlar

- [ ] Facility listesi açılır.
- [ ] Şirket/şube/durum/tür filtreleri çalışır.
- [ ] Serbest facility create yapılır.
- [ ] Şube Açılışı ile oluşan facility görünür.
- [ ] Facility detayında branch bağlantısı görünür.
- [ ] Reusable badge görünür.
- [ ] Facility scoped authority read-only görünür.
- [ ] Facility impact paneli aktif branch/authority uyarılarını gösterir.
- [ ] Teknik hata kullanıcıya gösterilmez.

## Seed Data

- Aktif şirket.
- Aktif şube.
- Kapalı şube.
- Root ve child organization unit.
- Aktif/reusable facility.
- Organization ve facility scope representative authority.
