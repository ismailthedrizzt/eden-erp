# UX Pattern Catalog

Date: 2026-06-06

## Canonical Product Patterns

- `Ekle` creates draft state.
- Lifecycle operations use wizard flows.
- Official data fields are not edited directly when an operation owns them.
- Document slots are contextual upload points.
- Central document management is for governance and reuse, not forcing users to search files before upload.
- Action Center presents business work, approvals, warnings and deadlines.
- Forms and wizards must share required-field standards.
- Final wizard submit labels should be consistent and endpoint-backed.

## Current Audit Status

The patterns exist in several modules but are not yet universal. Template consolidation remains a platform cleanup task.
