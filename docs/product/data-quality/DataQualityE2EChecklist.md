# Data Quality E2E Checklist

## Seeds

- Duplicate master organization
- Duplicate master person
- Cari baglantisi olmayan stakeholder
- Facility baglantisi eksik branch
- Zorunlu belgesi eksik employee
- Duplicate serial no installed asset

## E2E Basliklari

- Quality dashboard opens
- Duplicate detection run
- Duplicate group detail
- Merge preview
- Merge confirm
- Dismiss duplicate
- False positive mark
- Quality finding by entity
- Action Center quality warning
- Import duplicate warning
- Unauthorized merge denied

## Manuel Kontrol

1. Veri Kalitesi sayfasi acilir.
2. Duplicate detection calisir.
3. Duplicate group gorunur.
4. Merge preview acilir.
5. Relation impact gorunur.
6. Merge confirm calisir veya guvenli sekilde engeller.
7. False positive isaretlenir.
8. Quality score gorunur.
9. Import duplicate warning calisir.
10. Action Center quality warning gorunur.
11. Yetkisiz kullanici merge yapamaz.
12. Teknik hata kullaniciya gosterilmez.
