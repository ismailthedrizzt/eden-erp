# License Lifecycle

Statuses:

- `trial`: plan modules enabled.
- `active`: plan modules enabled.
- `past_due`: plan modules enabled with warnings.
- `suspended`: login may remain, module access is blocked/read-only by policy.
- `cancelled`: module access blocked.
- `expired`: module access blocked.
- `development`: development plan behavior.
- `internal`: internal/vendor behavior.
- `archived`: blocked and retained for history.

Operations:

- create license
- change plan
- suspend
- reactivate
- cancel
- archive
- record manual payment
- record usage snapshot
