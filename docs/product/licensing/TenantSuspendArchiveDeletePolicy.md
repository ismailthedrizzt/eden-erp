# Tenant Suspend, Archive and Delete Policy

Tenant deletion is dangerous.

## Suspend

- Temporary usage stop.
- Data is retained.
- Module access is blocked or read-only.

## Archive

- Tenant is outside active use.
- Data remains for reporting/backup.
- Normal users cannot enter.

## Hard Delete

- Allowed only for development/demo tenants.
- Requires explicit tenant-name confirmation.
- Requires platform/system admin permission.
- Customer/release tenants cannot be hard-deleted.
