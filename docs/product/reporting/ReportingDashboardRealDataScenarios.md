# Reporting Dashboard Real Data Scenarios

## Scenario 1 - Yonetim ana paneli

1. Kullanici dashboard acar.
2. Yetkili oldugu sirketlerin ozetini gorur.
3. Kapali modul kartlari gorunmez veya disabled gorunur.

## Scenario 2 - Ownership warning

1. Bir sirketin ownership toplami %100 degildir.
2. Dashboard warning karti gosterir.
3. Current ownership raporuna gider.

## Scenario 3 - Suresi yaklasan temsilci yetkileri

1. 30 gun icinde bitecek authority vardir.
2. Dashboard warning/critical karti gosterir.
3. Representative authority report acilir.

## Scenario 4 - Belge aranacak muhasebe hareketleri

1. Cari hareket document_status belge aranacak durumundadir.
2. Accounting KPI gosterir.
3. Missing documents report acilir.

## Scenario 5 - Geciken gorevler

1. Project task overdue olur.
2. Dashboard ve Action Center uyarir.
3. Gorev listesine gidilir.

## Scenario 6 - Bakimi gelen urunler

1. Installed asset next_maintenance_date yaklasir.
2. After-Sales KPI gosterir.
3. Maintenance due report acilir.

## Scenario 7 - Audit critical

1. Permission denied attempts vardir.
2. Admin dashboard audit/security card gosterir.

## Scenario 8 - Yetkisiz finansal KPI

1. Accounting permission olmayan kullanici dashboard acar.
2. Finansal kartlar gizli veya disabled olur.
