# After-Sales E2E Checklist

- Product create
- Product `after_sales_enabled`
- Installed asset create
- Serial required validation
- Warranty date calculation
- Service request create
- Service request assign
- Service record create
- Service record complete
- Maintenance due list
- Follow-up task creation
- Action Center integration through project task
- Unauthorized user blocked
- Technical errors hidden behind user-facing messages

## Seed Data

- Active company
- Customer cari account
- Serviceable product
- Installed asset
- Technician employee/user
- Open service request
