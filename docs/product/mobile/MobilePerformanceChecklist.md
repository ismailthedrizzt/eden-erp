# Mobile Performance Checklist

- Dashboard bolumleri lazy load edilebilir olmali.
- Grafikler agirsa route-level veya component-level lazy render edilmeli.
- SmartDataTable server-side pagination kullanmali.
- Mobilde tablo yerine kart render edildigi icin yatay overflow azaltilmali.
- Buyuk dosya/fotograf preview boyutu sinirlanmali.
- Service worker API data cache yapmamali.
- Static asset cache disindaki hassas data cache disi kalmali.
- `npm run build` sonrasinda bundle uyarilari izlenmeli.
- Lighthouse PWA/Performance smoke test gelecek fazda eklenmeli.
